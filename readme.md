## Installation

1. Clone or download repository
2. Install dependencies
  ```sh
  // with npm
  npm install
  
  // with yarn
  yarn install
  ```
3. Copy env.example to .env and set port.
2. Start server
  ```sh
  // with npm
  npm run watch-server
  
  // with yarn
  yarn run watch-server
  ```
